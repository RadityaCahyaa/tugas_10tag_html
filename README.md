<html>  
Fungsi:  
Tag <html> merupakan elemen utama yang membungkus seluruh isi halaman web agar dapat dikenali sebagai dokumen HTML oleh browser.

1. <head>  
Fungsi:  
Tag <head> digunakan untuk menyimpan informasi penting tentang halaman seperti metadata, judul, dan link ke file eksternal (CSS/JavaScript).

2. <title>  
Fungsi:  
Tag <title> berfungsi untuk menentukan judul halaman yang tampil pada tab browser.

3. <body>  
Fungsi:  
Tag <body> berisi semua konten yang ditampilkan kepada pengguna, seperti teks, gambar, dan link.

4. <h1>  
Fungsi:  
Tag <h1> digunakan untuk membuat judul utama dengan ukuran paling besar.

5. <p>  
Fungsi:  
Tag <p> digunakan untuk membuat paragraf agar teks lebih rapi dan mudah dibaca.

6. <a>  
Fungsi:  
Tag <a> digunakan untuk membuat tautan atau link ke halaman lain.

7. <img>  
Fungsi:  
Tag <img> digunakan untuk menampilkan gambar pada halaman web.

8. <ul> dan <li>  
Fungsi:  
Tag <ul> digunakan untuk membuat daftar tidak berurutan, sedangkan <li> digunakan sebagai isi dari daftar tersebut.

9. <br>  
Fungsi:  
Tag <br> digunakan untuk membuat baris baru atau pindah ke baris berikutnya tanpa membuat paragraf baru.

10. <button>  
Fungsi:  
Tag <button> digunakan untuk membuat tombol yang bisa diklik oleh pengguna, biasanya digunakan untuk menjalankan aksi seperti mengirim data atau menjalankan fungsi tertentu.
